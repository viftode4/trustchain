# Commands

## Build & Test
- `cargo build --workspace` — build all crates
- `cargo test --workspace` — run all tests (224 tests)
- `cargo clippy --workspace -- -D warnings` — lint (must pass clean)
- `cargo fmt --all -- --check` — format check

## Run
- `cargo run --bin trustchain-node -- launch --name X` — start a node

## System (Windows/bash)
- `git`, `ls`, `cd`, `grep`, `find` — standard unix via bash shell
